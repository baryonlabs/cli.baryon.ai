한 줄 설치로
AI 코딩·학습
에이전트.

브라우저 Web UI, 코드를 직접 쓰고 실행하는 코딩 에이전트(pi), 그리고 이를 구동하는 baryon.ai API를 하나로 묶었습니다. LLM API까지 Baryon이 직접 제공 — 기관의 별도 단기 API 계약이 필요 없습니다.

terminal — baryon cli
$npm install -g @baryonlabs/cli
$baryon
✔ baryon.ai API 기본 연결됨 · 모델 비교/전환 준비 완료
✔ 세션 · 메모리 · 도구 호출 · 서브에이전트 활성화
설치 가이드 보기 Web UI 바로 접속 ↗
Node.js ≥ 22 만 있으면 끝 Docker 불필요 · 빌드 과정 없음 상용·로컬 모델 무중단 전환
Quick Start

두 경로로 즉시 시작.

학생·교강사는 설치 없이 브라우저로, 코딩 실습은 명령 한 줄로. 두 경로 모두 baryon.ai API가 기본 백엔드로 연결됩니다.

1

설치

npm install -g @baryonlabs/cli — pnpm · yarn · bun 모두 지원. Node.js만 있으면 완료됩니다.

2

연결

baryon setup — vibecamp.us 에서 발급한 키(vc_live_…)를 한 번 등록하면 ~/.pi/agent/models.json에 baryon 프로바이더가 자동 구성됩니다. baryon keys 로 발급 페이지를 바로 엽니다.

3

실행

baryon 으로 대화형 코딩 에이전트 시작. baryon -p "..." 로 단발 실행(JSON 출력)도 가능합니다.

4

전환

baryon models 로 사용 가능한 모델을 보고, 세션 중 /model 로 상용·로컬 모델을 자유롭게 비교·전환합니다.

~/project — baryon
$baryon setup
baryon.ai API key › ••••••••
✔ wrote ~/.pi/agent/models.json
✔ provider baryon · 4 models
$baryon -p "CSV 분석해 차트 만들어줘"
→ 코드 작성 · 실행 · 산출물 생성…
✔ report.html · chart.svg 생성됨
Architecture

세 계층, 단일 패키지.

Web UI · 코딩 에이전트 · LLM 백엔드 — 세 계층 모두 함께 배포됩니다. 별도 구축 인력이 필요 없습니다.

LAYER 01

Web UI

chat.baryon.ai · pi-web-ui

여러 LLM을 비교·전환하는 브라우저 인터페이스. 세션·메모리, 도구 호출, 문서 생성을 화면에서 직접 관찰. 만든 HTML은 miri.dev로 즉시 공개 URL 게시. PDF·DOCX·XLSX·PPTX·이미지 첨부 지원.

LAYER 02

코딩 에이전트 (pi)

@baryonlabs/cli → pi

코드로 문서를 생성하고 자료를 분석하는 에이전트. 서브에이전트로 작업을 분해·위임·통합. 대화형 / 출력(JSON) / RPC / SDK 4가지 모드, 세션 분기·재개, TypeScript 확장 지원.

LAYER 03

LLM 백엔드

baryon.ai API · OpenAI 호환

에이전트와 UI를 구동하는 모델 접속 계층. 별도 키 설정 없이 패키지에 내장 제공. 필요 시 로컬 LLM(Ollama·LM Studio·vLLM) 또는 기관 자체 키로 무중단 전환.

사용자(브라우저) → Web UI → 코딩 에이전트 (pi) → baryon.ai API (또는 로컬 LLM) → 결과 반환

→ 오케스트레이션
Multi-Provider

하나의 인터페이스,
모든 모델.

기본값은 baryon.ai API. 기관 정책에 따라 상용 모델 또는 완전 오프라인 로컬 모델로 무중단 전환합니다. 폐쇄망·개인정보 민감 환경에 특히 중요한 운영 옵션입니다.

baryon.ai (기본) Anthropic OpenAI Google Gemini Azure OpenAI OpenRouter Groq Ollama · 로컬 LM Studio · 로컬 vLLM · 로컬
Deployment

클라우드부터 폐쇄망까지.

교내 환경에 맞춰 4가지 배포 형태를 지원합니다. 온프레미스·오프라인 구동 시 학생 코드·과제 데이터가 외부로 나가지 않습니다.

클라우드 (SaaS) FASTEST

Baryon 호스팅. 가장 빠른 도입, 별도 인프라 불필요. 인터넷 연결 환경.

온프레미스 GOVERNANCE

기관 서버·교내 워크스테이션에 설치. 데이터 외부 미전송, 거버넌스 강화.

오프라인 (로컬 LLM) AIR-GAP

폐쇄망에서 로컬 모델(Ollama·LM Studio·vLLM)로 완전 오프라인 구동.

엣지 / 현장 설치 EDGE

교내 서버 또는 GPU 워크스테이션 단위 설치, 다중 강의실 동시 운영.
